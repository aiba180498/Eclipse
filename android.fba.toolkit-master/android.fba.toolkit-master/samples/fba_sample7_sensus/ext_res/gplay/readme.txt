-============= Название =============- 
1С:Cенсус территории

-============= Краткое описание =============- 
Перепись торговых точек на территории. Приложение для мобильной торговли, где в процессе осуществления продажи задействованы выездные сотрудники 
(торговые представители, мерчендайзеры, супервайзеры, банковские и страховые агенты и прочие).


-============= Полное описание =============- 
Приложение демонстрирует возможности фреймворка 'FBA Toolkit' для создания клиент-серверных приложений 1С+Android.

FBA Toolkit это:
•платформа для корпоративных мобильных приложений
•разработка «нативного» приложения Android в родной IDE на языке Java
•1С: Предприятие 8.2 (8.3) в роли источника данных, корпоративной информационной системы
•доступ ко всем возможностям платформы Android: высокоскоростные протоколы связи Wi-fi, 3G, 4G; карты; средства мониторинга, позиционирования и контроля; виджеты; мультимедиа, интеграция с онлайн-сервисами и прочее
•бесплатное ПО с открытым исходным кодом

Данное ДЕМОНСТРАЦИОННОЕ приложение предназначено для проведения сенсуса (retail census) – полной переписи всех точек розничной торговли на определенной территории.

Функционал:
•отображение списка торговых точек, редактирование контактной информации по ним (наименование, телефон, сайт, статус, комментарий)
•фото-отчет по торговой точке
•отображение точки на карте, текущая позиция агента, навигация по маршруту (используется Google Maps)
•добавление новой торговой точки в базу (определение текущих координат, определение адреса по координатам, создание основного фото)
•отображение списка задач к исполнению (дата начала, дата исполнения, комментарий). Интеграция с эко-системой Android: задача может быть сразу добавлена в календарь.
•обмен данными с информационной системой 1С (по требованию; в автоматическом режиме по расписанию)

Синхронизация с сервером 1С производится в режиме симуляции, используется тестовый набор данных – пиццерии в центре г. Орел. Для первоначальной загрузки данных выполните обмен по варианту «Начальная инициализация».

Внешний вид программы выдержан в стиле новой дизайнерской философии Material Design с целью доказать, что и корпоративные мобильные решения могут быть современными, удобными и функциональными.

Исходный код приложения доступен на https://github.com/avesha/android.fba.toolkit/tree/master/samples/fba_sample7_sensus
